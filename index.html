Grouping Content

The 'div' (short for "division") element is a block-level container, which means it starts on a new line and takes up the full width available.

Span

The 'span' element is an inline container, which means it does not start on a new line and only takes up as much width as necessary. It is used to apply styles or manipulate a specific part of the content without affecting the layout.
