Text-Level Semantic Elements

Time

The 'time' element represents a specific period in time. This can be a time on a 24-hour clock, a precise date in the Gregorian calendar, or both.

The event starts at 10:00 AM.

The project deadline is December 31, 2024.


Mark

This text is highlighted to draw attention.


abbr

The 'abbr' element represents an abbreviation or acronym, with an optional title attribute to provide the full term.

WHO was founded in 1948.


City

The 'cite' element represents the title of a work, such as a book, a movie, a research paper, or an artistic work.

The Great Gatsby by F. Scott Fitzgerald is a classic novel.


Code

The 'code' element represents a fragment of computer code.

To print "Hello, World!" in Python, use print("Hello, World!").
